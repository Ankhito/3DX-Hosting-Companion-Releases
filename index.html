Good to know

Before you install.

Is this a 3DX mod?+

No. It is an external Windows companion. It does not inject into, patch, or modify 3DXChat.

How does sending work?+

The app detects the visible 3DX chat input and uses brief Windows mouse, clipboard, and keyboard simulation. Your cursor is restored after sending.

Why might Windows show a warning?+

The application is not yet code-signed. Download only from this official release repository and verify the included SHA-256 checksum if desired.

Do I need a Google or Discord account?+

No. Accounts are optional and currently prepare the app for future cloud services. Every local hosting feature remains available while signed out or offline.

Does it support Venue Host or Greeter mode?+

Venue Host and Greeter are visible in the Hosting Mode switcher as future modes, but neither is included in the current release.
