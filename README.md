# 3DX Hosting Companion Releases

Official public website and Windows installer releases for **3DX Hosting Companion**.

Version 0.5.6 improves countdown-send reliability and makes custom-status management clearly accessible beside the player count. Optional Google and Discord accounts remain available while every local hosting tool continues to work without signing in.

Visit the [3DX Hosting Companion website](https://ankhito.github.io/3DX-Hosting-Companion-Releases/) for an overview, setup flow, and the latest download.

Use the [latest release](../../releases/latest) to download:

- `3DX.Hosting.Companion.Setup.msi`
- `3DX.Hosting.Companion.Setup.msi.sha256`

Read the [styled public patch notes](https://ankhito.github.io/3DX-Hosting-Companion-Releases/changelog.html) to see what changed in each version. The underlying [Markdown changelog](CHANGELOG.md) remains available for reference.

The application source code is maintained in a separate private repository. This repository contains no source code.
