01
FIRST RUN

Find the visible 3DX chat.

Open 3DX before the Companion. A green detection label confirms the game is available. The startup scan locates the movable chat input and caches it.

  • Use Scan again after moving chat.
  • Choose silent rescans every 0, 5, or 10 minutes.
  • The app remains external and restores your cursor after sending.

PHRASE LIBRARY

One folder for every board.

The compact library tree keeps boards and phrases easy to browse. Select a folder to see only its phrases, then search within it.

  • Create, rename, import, and export beside the tree.
  • Moving a parent phrase moves its children by default.
  • Import individual phrases from PhraseExpress in the editor.

03
PHRASE EDITOR

Create, preview, and connect.

Name the phrase, write its message, choose a folder, and save. The live preview approximates 3DX Local Chat.

  • Insert colored native 3DX emojis from five categories.
  • Add an optional text-expansion command such as !welcome.
  • Link child outcomes to a parent rule and give them roll numbers.
  • Every message follows the 400-character chat limit.

04
OVERLAYS

Keep the play field clear.

The Phrase Overlay becomes a responsive button board. The separate Turn / Skip Overlay keeps live game controls together.

  • Switch folders using the arrow beside the active folder.
  • Parent buttons open a compact child-outcome menu.
  • Enable Quickbind, hover a phrase, and press a key; Esc clears it.
  • Resize freely and choose independent opacity in Settings.

05
TURN ORDER

Manage a fluid player list.

Type a name and press Enter. Drag cards to reorder live, change built-in or custom statuses, randomize, or remove players with the bin icon.

  • Open Manage Statuses beside the player count to create short markers such as SH for Shield.
  • Choose arrows or commas for Turn Order output.
  • Send Player Order and Turn Order separately.
  • Edit visual, emoji-aware Pause and Resume messages under Host messages.
  • Select a player for a protected 5- or 10-second skip countdown.

06
ROOM FILES

Keep room files safe and close.

Select the Room Files you use, keep them local, or manually create an optional account-backed cloud copy.

  • Room Files remain local unless you explicitly start an optional cloud backup.
  • Cloud copies preserve the exact bytes and are checksum-verified before restore.
  • Remove shortcuts without deleting the source file.
  • References are included in local backups.

07
SETTINGS & BACKUP

Make it fit your setup.

Settings are stored locally and remain available without an account.

  • Configure shortcuts, opacity, headers, and silent rescans.
  • Create or restore a portable local backup.
  • Check for verified installer updates inside the app.
  • Google and Discord accounts are optional and needed only for cloud features.
  • Signing in alone never uploads phrases or Room Files.
