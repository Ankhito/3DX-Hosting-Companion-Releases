Privacy Policy

3DX Hosting Companion 1.0 is a local application with no Companion account or cloud-storage service.

Data stored locally

Phrases, folders, commands, quickbinds, turn-order state, overlay settings, application settings, local backups, and diagnostic logs are stored on your computer under the Companion application-data folder. The app does not upload this content to a Companion service.

Network access

The app contacts GitHub Releases to check for application updates and download an installer only when an update is selected. The Ko-fi button opens the developer's Ko-fi page in your browser. Those services process requests under their own privacy policies.

Feedback

Feedback remains welcome through the public release repository. Information you post to GitHub is public and governed by GitHub's privacy terms. Do not include private phrases, logs, or personal information in a public report.

Children

The Companion is intended only for adults eligible to use 3DXChat.

Changes and contact

Material changes will appear here and in public patch notes. Questions can be raised through the public release repository.
